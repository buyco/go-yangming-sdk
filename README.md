- [x] Tracking
- [ ] Schedules
- [ ] Booking

## How to install Go ?

#### Debian / Ubuntu:

```bash
$ sudo apt update
$ sudo apt install golang
```

#### Arch:

```bash
$ sudo pacman -Sy go
```

#### Mac OS X:

```bash
$ brew update
$ brew install golang
```

#### Last release from script:

See: https://github.com/udhos/update-golang

#### From tarballs:

See: https://golang.org/doc/install

## Check golang version

```bash
$ go version
```

## Commands:

Available commands:

```bash
$ make help
```

SDK creation
